Manual do Usuário - Formulário de Match


Bem-vindos ao nosso formulário de match! Com este formulário, você pode inserir suas informações pessoais e as informações do seu crush para descobrir o quão compatíveis vocês são. Por favor, siga as instruções abaixo para preencher o formulário corretamente.

Passo 1: Insira suas informações pessoais Nome: Digite seu nome no campo Nome. Este campo é obrigatório, então certifique-se de preenchê-lo. Idade: Insira sua idade no campo Idade. Este campo também é obrigatório e deve ser um número maior ou igual a zero. Gênero: Selecione seu gênero no menu suspenso "Gênero". Este campo é obrigatório, então escolha a opção que melhor se aplica a você.

Passo 2: Insira os dados do seu crush Nome do Crush: Digite o nome do seu crush no campo Nome do Crush. Este campo é obrigatório. Idade do Crush: Insira a idade do seu crush no campo Idade do Crush. Este campo é obrigatório e deve ser um número maior ou igual a zero. Gênero do Crush: Selecione o gênero do seu crush no menu suspenso Gênero do Crush. Este campo é obrigatório.

Passo 3: Enviar o formulário Depois de preencher todos os campos, clique no botão Enviar para enviar o formulário. Certifique-se de que todas as informações estão totalmente erradas corretas antes de enviar.
